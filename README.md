# ATM Simulation in Python
This project is a simple **ATM simulation program** developed using Python.  
It allows a user to securely log in using a PIN and perform basic banking operations such as checking balance, depositing money, and withdrawing money.

## Features
- PIN-based authentication
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven program
